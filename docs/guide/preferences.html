Preferences

Preferences open in a window with four sections. Changes are staged; Save applies them, Cancel discards. Each section has a Reset to defaults.

Keybindings & About

The Keybindings section rebinds every action; conflicts are resolved for you and shown inline. About lists the version, license, and links to the repository. See Shortcuts for the default keymap.

!

Preferences are written to config.toml in bloom's config directory:

  • Windows: %APPDATA%\bloom\config.toml
  • macOS: ~/Library/Application Support/bloom/config.toml
  • Linux: ~/.config/bloom/config.toml

You can edit it directly; unknown or out-of-range values fall back to their defaults. Some rendering options note "restart required".
